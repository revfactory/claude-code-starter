# Claude Code 시작하기

> AI와 함께 코딩하는 새로운 방법 — Claude Code 입문서

설치부터 고급 기능까지, 단계별로 Claude Code의 모든 것을 다루는 한국어 입문서입니다.

## 대상 독자

- 프로그래밍 기초는 있지만 AI 코딩 도구가 처음인 개발자
- 다른 AI 도구(Copilot 등)를 써봤지만 Claude Code는 처음인 개발자
- 팀에 Claude Code 도입을 검토하는 테크 리드

## 읽는 방법

**처음부터 순서대로** 읽는 것을 권장합니다. 특히 Part 1(챕터 1~3)은 반드시 먼저 읽어주세요.

챕터 3까지 읽었다면 관심 있는 챕터로 바로 이동해도 됩니다:
- Git 워크플로우가 궁금하다면 → 챕터 8
- CLAUDE.md 설정이 궁금하다면 → 챕터 9
- 비용이 걱정된다면 → 챕터 11
- 프롬프트를 잘 쓰고 싶다면 → 챕터 16

## 제공 형식

### 마크다운 원본
`book/` 디렉토리에 챕터별 마크다운 파일이 있습니다.

```
book/
├── 00_목차.md
├── 01_claude_code란_무엇인가.md
├── ...
├── 18_부록.md
└── images/          # 인포그래픽 이미지
```

### PDF
`book/Claude_Code_시작하기.pdf` — 전체 134페이지, 한국어 폰트(Noto Sans KR) 적용.

### 웹 뷰어
`book/web/index.html` — 페이지 넘기기 효과가 있는 브라우저 기반 뷰어.

```bash
# 로컬에서 웹 뷰어 실행
cd book/web
python -m http.server 8080
# 브라우저에서 http://localhost:8080 접속
```

- 데스크톱: 양면 펼침 + 클릭/키보드(←→)로 페이지 넘기기
- 모바일: 단면 + 터치 스와이프로 페이지 넘기기
