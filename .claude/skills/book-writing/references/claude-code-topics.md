# Claude Code 도서 주제 레퍼런스

## 핵심 주제 영역

### 1. 기초 (Getting Started)
- Claude Code란 무엇인가
- 설치 및 초기 설정
- 첫 번째 대화: Hello World
- 터미널 기본 (CLI에 익숙하지 않은 독자를 위해)
- 인증 및 API 키 설정

### 2. 일상 사용 (Daily Usage)
- 코드 질문하기 (코드 설명, 디버깅)
- 코드 수정 요청하기
- 파일 읽기/쓰기/편집
- 검색 (Glob, Grep)
- 터미널 명령 실행
- 멀티턴 대화로 작업하기

### 3. 프로젝트 설정 (Project Configuration)
- CLAUDE.md 작성법
- 프로젝트별 설정
- .claude/ 디렉토리 구조
- 권한 모드 (plan, auto, bypassPermissions 등)

### 4. 고급 기능 (Advanced Features)
- MCP 서버 연동
- 커스텀 슬래시 커맨드
- 훅(Hooks) 시스템
- 에이전트 정의 및 팀 구성
- 스킬 시스템
- IDE 통합 (VS Code, JetBrains)

### 5. 워크플로우 (Workflows)
- Git 워크플로우 (커밋, PR, 코드리뷰)
- TDD 워크플로우
- 리팩토링 워크플로우
- 디버깅 워크플로우
- 문서화 워크플로우

### 6. 베스트 프랙티스 (Best Practices)
- 효과적인 프롬프트 작성법
- 컨텍스트 관리
- 대규모 프로젝트에서의 사용
- 보안 고려사항
- 비용 최적화

### 7. 실전 프로젝트 (Hands-on Projects)
- 웹 앱 만들기 (처음부터 끝까지)
- API 서버 구축
- CLI 도구 만들기
- 기존 프로젝트에 Claude Code 도입하기

## 독자 페르소나
- **초보 개발자**: 프로그래밍 기초는 있지만 AI 도구 경험 없음
- **경험 있는 개발자**: 다른 AI 도구(Copilot 등)는 써봤지만 Claude Code는 처음
- **비개발자 테크 리드**: 팀에 도입 검토 중
