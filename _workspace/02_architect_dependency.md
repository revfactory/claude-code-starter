# 챕터 간 의존성 맵

## 1. 의존성 다이어그램

```
Part 1: 시작하기
  CH01 (Claude Code란) ──→ CH02 (설치) ──→ CH03 (첫 대화)
                                              │
                    ┌─────────────────────────┘
                    │
Part 2: 일상 사용법  │
                    ▼
                  CH04 (터미널 인터페이스)
                    │
          ┌─────────┼─────────┐
          ▼         ▼         ▼
        CH05      CH06      CH07
      (코드 읽기) (코드 수정) (명령/테스트)
          │         │         │
          └─────────┼─────────┘
                    ▼
                  CH08 (Git 워크플로우)

Part 3: 프로젝트 설정
                  CH09 (CLAUDE.md) ←── CH03 이후 독립 진입 가능
                    │
              ┌─────┴─────┐
              ▼           ▼
            CH10        CH11
          (권한/보안)  (모델/비용)

Part 4: 고급 기능
            CH12 (컨텍스트 관리) ←── CH04 이후 독립 진입 가능
            CH13 (커맨드/스킬)  ←── CH09 선행 권장
            CH14 (MCP 서버)    ←── CH03 이후 독립 진입 가능
            CH15 (훅/에이전트)  ←── CH10 선행 권장

Part 5: 실전 워크플로우
            CH16 (프롬프트 작성) ←── CH03 이후 독립 진입 가능
            CH17 (개발 워크플로우) ←── CH05~CH08 선행 권장
```

## 3. 독립 읽기 가능 챕터

다음 챕터는 CH03(첫 대화)만 읽었다면 바로 진입할 수 있습니다:

- **CH05**: 코드 읽기와 검색
- **CH09**: CLAUDE.md
- **CH10**: 권한과 보안
- **CH11**: 모델 선택과 비용 관리
- **CH14**: MCP 서버 연동
- **CH16**: 효과적인 프롬프트 작성법

## 4. 병렬 집필 가능 그룹

### 그룹 A: 기반 (순차 집필 필수)
```
CH01 → CH02 → CH03
```
- 가장 먼저 완성해야 하는 핵심 경로
- 1명의 저자가 순차 집필 권장

### 그룹 B: 일상 사용법 (CH03 완성 후 병렬 가능)
```
CH04  CH05  CH06  CH07
```
- CH04~CH07은 서로 약한 참조만 있으므로 병렬 집필 가능
- 단, CH08(Git)은 CH06, CH07 이후 집필 권장

### 그룹 C: 프로젝트 설정 (CH03 완성 후 병렬 가능)
```
CH09  CH10  CH11
```
- 3개 모두 독립적으로 집필 가능
- CH09는 후속 챕터(CH12, CH13, CH15)의 기반이므로 우선 완성 권장

### 그룹 D: 고급 기능 (그룹 B, C 부분 완성 후 병렬 가능)
```
CH12  CH13  CH14  CH15
```
- 4개 모두 서로 독립적이므로 병렬 집필 가능
- 각각 다른 선행 챕터를 참조하므로 해당 선행만 완성되면 시작 가능

### 그룹 E: 실전 (그룹 B 완성 후)
```
CH16  CH17
```
- CH16은 독립 집필 가능
- CH17은 Part 2 전체(CH05~CH08) 완성 후 집필

### 그룹 F: 부록 (본문 전체 완성 후)
```
부록 A  부록 B  부록 C  부록 D
```
- 4개 모두 병렬 집필 가능
- 본문 내용 확정 후 작성

## 5. 권장 집필 순서 (크리티컬 패스)

```
Phase 1: CH01 → CH02 → CH03                    [기반]
Phase 2: CH04, CH05, CH06, CH09 (병렬)          [핵심 일상 + 설정]
Phase 3: CH07, CH08, CH10, CH11 (병렬)          [일상 완성 + 설정 완성]
Phase 4: CH12, CH13, CH14, CH15, CH16 (병렬)    [고급 + 프롬프트]
Phase 5: CH17                                   [종합 실전]
Phase 6: 부록 A~D (병렬)                         [레퍼런스]
```
